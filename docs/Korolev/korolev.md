### Starting Up

After cloning the repo. In a terminal in the repo.
You may use npm or pnpm

```bash
$ npm install
```

### Run
```bash
$ npm run dev
# for the SCSS compilers (global/atomic)
$ npm run sass:global
# build
$ npm run build
```

Go to Point your web browser to [http://localhost:5000](http://localhost:5000)
